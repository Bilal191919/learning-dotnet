# Logic Building: Dynamic FizzBuzz Game (.NET)

A professional console-based application that demonstrates clean coding practices and runtime logic extension.

## Features
- **Dynamic Divisors:** Users can add custom rules at runtime.
- **Error Handling:** Robust validation for user inputs.
- **Standardized Structure:** Follows professional .NET project conventions.

## How to Run
1. Open `Stage-1/FizzBuzz.slnx` in Visual Studio.
2. Press `F5` to run the application.
